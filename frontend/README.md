# mcp-aws-sa landing

Next.js 14 (App Router) + Tailwind CSS landing page for [mcp-aws-solution-architect](../).

## Local dev

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:3000.

## Deploy on Vercel

1. Import the repo on vercel.com.
2. **Root Directory:** `frontend/`
3. Framework preset: Next.js (auto-detected).
4. No environment variables needed.

Production: https://mcp-aws-solution-architect.vercel.app (after deploy).
